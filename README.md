# Fake News Headline Generator

## Overview
A Python-based application that generates fake news headlines using predefined datasets. 
This project is designed to demonstrate core Python concepts and logical problem-solving.

## Features
- Random headline generation
- Multiple categories (Politics, Sports, Technology, Entertainment)
- Simple and readable code structure

## Technology Stack
- Python 3

## How to Execute
1. Clone the repository
2. Run the file using:
   python main.py

## Key Learnings
- Python control structures
- Random module usage
- String manipulation
- Basic application design
